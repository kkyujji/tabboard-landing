# 탭보드 랜딩페이지 이미지 적용 버전

실제 화면 이미지 3장을 Hero 목업에 배치한 버전입니다.

- `index.html`: 페이지 구조와 이미지 배치
- `styles.css`: 반응형 레이아웃과 이미지 크기
- `data.js`: 기능 카드, 활용 카드, FAQ 문구
- `app.js`: 카드 출력과 FAQ 아코디언 동작
- `assets/tabboard-logo.png`: 탭보드 로고
- `assets/timeline.png`: 세로형 타임라인 화면
- `assets/poll.png`: 투표 결과 화면
- `assets/drawing.png`: 그리기 도구 화면

이미지 배치는 타임라인을 상단 큰 카드로, 투표와 그리기를 하단 2분할 카드로 구성했습니다.
